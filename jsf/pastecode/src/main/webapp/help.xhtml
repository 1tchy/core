Useful Information

Paste Code:
  • Enter your nick - This is optional, if it is not entered the code will be listed under "Anonymous" user
  • Choose a language of your code - This way you can define formatting of code after submitting the form
  • Choose if this code is private - When this option is chosen the application will generate a hash value of your code and will provide you with a unique link (something like http://localhost:8080/weld-pastecode/582ad4b5ed409ec8b81c6e33ee8b72377890bc86). Furthemore your code will not be listed in the right panel and cannot even be found through a search capability.
  • Enter your code and a note you would like to send with the code
  • Submit the form and forward the resulting link
Search Code:
  • Go to a history page - Now you can see all the posts with pagination
  • Enter data for searching - You can combine all the data together.
Weld Features Covered:
  • Injecting into POJO, EJB (SFSB), Servlet
  • Annotations @ApplicationScoped, @Model, @SessionScoped et al.
  • Producer named methods
  • Decorators - This feature is used to log transactions during which a code larger than 65kB is stored to a database


Enjoy work with WELD !
